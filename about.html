About

Seasonal 365-Day Tree Calendar

A beautiful interactive 3D visualization where a tree represents the entire year, with 365 leaves representing each day. Each leaf changes color based on whether its day is in the past, today, or the future. The tree features four seasonal branches (Spring, Summer, Autumn, Winter) with distinct color palettes, creating an immersive and intuitive way to visualize time.
