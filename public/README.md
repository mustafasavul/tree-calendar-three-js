# Background Image

Place your panoramic background image here as `bg.jpg`.

**Recommended specifications:**
- Format: JPG or PNG
- Aspect ratio: 16:9 (e.g., 1920x1080, 3840x2160)
- Content: Japanese historic garden / temple area (or any scenic panoramic view)
- Size: Optimized for web (under 2MB recommended)

**Note:** If `bg.jpg` is not provided, the application will use a gradient sky blue background instead.
